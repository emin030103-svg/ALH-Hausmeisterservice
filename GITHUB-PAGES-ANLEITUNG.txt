ALH Hausmeisterservice - GitHub Pages Upload

Variante A: GitHub Website
1. Neues GitHub Repository erstellen, z. B. alh-hausmeisterservice.
2. Alle Dateien aus diesem Paket direkt in das Repository hochladen.
3. In GitHub: Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main, Ordner: /root auswaehlen und speichern.

Variante B: Eigene Domain alh-hausmeisterservice.de
Die Datei CNAME ist bereits enthalten.
Danach muss die Domain beim Anbieter per DNS auf GitHub Pages zeigen.

Wichtig: index.html muss direkt im Hauptverzeichnis des Repositorys liegen.
